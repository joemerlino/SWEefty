\lettera{J}

\parola{Java}{Linguaggio di programmazione orientato agli oggetti fortemente tipizzato.}
\parola{Javascript}{Linguaggio di scripting utilizzato principalmente per applicazioni web.}
\parola{JSON}{\emph{JavaScript Object Notation}, formato di file standard che utilizza un testo comprensibile dall'uomo per trasmettere dati costituiti da coppie valore-attributo.}
