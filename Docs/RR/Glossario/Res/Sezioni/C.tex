\parola{camel Case}{otazione usata per scrivere parole composte o frasi unendo tutte le parole tra loro, eliminando quindi gli spazi. Trova profondo nei linguaggi di programmazione in quanto nei nome delle variabili o delle funzioni non sonon concessi spazi.}
