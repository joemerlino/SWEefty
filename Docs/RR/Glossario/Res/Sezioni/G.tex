\lettera{G}

\parola{git}{È un software per il controllo di versione distribuito con interfaccia a riga di comando sviluppato da Linus Torvalds nel 2005.}
\parola{GitHub}{È un servizio web di hosting per progetti software che utilizzano Git come sistema di versionamento.}
\parola{GitKraken}{È un client desktop crossplatform per Git. Fornisce un interfaccia grafica per semplificare l'utilizzo di Git.}
\parola{GNU Aspell}{È un software open-source per il controllo ortografico di testi.}
\parola{Google Cloud Platform}{suite di servizi di cloud computing offerta da Google che funziona sulla stessa infrastruttura che Google utilizza internamente per i suoi prodotti per l'utente finale.}
\parola{Google SQL}{prodotto che fa parte della suite Google Cloud Platform che fornisce un database relazionale in un ambiente completamente gestito, che elimina l'onere di essere installato e impostato dall'utente.}
