\parola{MongoDB}{ MongoDB è un Database Managment System  non relazionale, orientato ai documenti libero ed open source. Classificato come un database di tipo NoSQL, MongoDB non si basa su tabelle come per i  database relazionali ma si bassa su documenti BSON, rendendo l'integrazione di dati di alcuni tipi di applicazioni più facile e veloce. }
